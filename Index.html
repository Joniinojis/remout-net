Библиотека документации ZyXEL

Версия 1.2 июнь 2002 года

Этот диск содержит библиотеку документации на русском языке по профессиональному оборудованию компании ZyXEL Communications Corporation, систему управления NetaVista для AES-100, IES-1000, VES-1000, а также драйвера и микропрограммы для модемов серии U-336.

Для удобства просмотра и распечатки документы подготовлены в формате Adobe PDF. Для работы с ними вам понадобится программа Adobe Acrobat Reader, которую можно установить с этого диска или получить с сервера Adobe.

Новые версии микропрограмм для оборудования ZyXEL и инструкции по их установке вы можете найти на сервере Московского представительства в разделе "Новые микропрограммы, файлы настройки и документация".

Драйвера для модемов серии U-336 под операционные системы Windows 95,NT,98,2000,ME и XP находятся в корне диска.

Мы рекомендуем вам зарегистрировать ваш модем через Интернет. Регистрация через Интернет дает дополнительный год бесплатной гарантии и ряд других преимуществ.

Дополнительную информацию можно найти на сервере Московского представительства ZyXEL. По вопросам технической поддержки обращайтесь по адресу support@zyxel.ru
